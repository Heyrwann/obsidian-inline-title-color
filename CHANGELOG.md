# Changelog

Toutes les modifications notables seront documentées ici.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/).

## [1.0.0] - 2026-01-31

### Ajouté

- Coloration du titre inline via la propriété YAML `inline-title-color`
- Support des formats de couleur : hex, rgb, rgba, noms CSS
- Validation automatique des codes couleur
- Mise à jour en temps réel lors du changement de fichier
- Mise à jour en temps réel lors de la modification des propriétés YAML

### Notes

- Première version stable
- Testé avec Obsidian v0.15.0+
